Extended Method
To have extended method first add declaration file and declare a method you want to extend: 
      
        interface Math {{ '{' }}
              randomInt(min: number, max: number): number;
        {{ '}' }}
Then in polyfills.ts implement your declared method: 
      
        Math.randomInt = function (min: number, max: number): number {{ '{' }}
          return Math.round(Math.random() * max + min);
        {{ '}' }};
You can even change prototype method implemenation in polyfills: 
      
        Date.prototype.toString = function (): string {{ '{' }}
          return 'new to string';
        {{ '}' }};
